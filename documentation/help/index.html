Гипертекстовый редактор


Гипертекстовый редактор - программа позволяющая создавать и редактировать материалы с гипертекстовой разметкой, удобной для создания справочных систем программного обеспечения.


Основные функции


Гипертекстовый редактор позволяет:


  • выполнять редактирование файлов с гипертекстовой разметкой, а также поддерживает обычные текстовые файлы и файлы с разметкой Markdown


  • производить форматирование текста (стиль шрифта, выравнивание, отступы, изменять цвет шрифта и фона текста)


  • вставлять гиперссылки и изображения с преобразованием абсолютных путей в относительные


  • сохранять файлы в форматах: HTML, TXT, MD, ODT и PDF


  • редактировать несколько файлов одновременно с помощью вкладок


  • работать с буфером обмена и сочетаниями горячих клавиш


  • получать быстрый доступ к файлам с помощью файлового менеджера


  • выполнять печать файлов. Поддерживает возможность предпросмотра перед печатью


  • выполнять поик по тексту (возможно задавать чувствительность к регистру, указывать направление поиска)


  • переключать интернационализацию (русский и английский)


  • поддерживает светлую и тёмную темы
