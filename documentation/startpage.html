Гипертекстовый нейроредактор


Применение генеративных LLM


Гипертекстовый нейроредактор - программа для отработки применения больших языковых моделей LLM для генерации контента. Примененяются модели доступные через API: GigaChat и YandexGPT. Позднее возможно использование моделей от OpenAI (через VPN). Необходимо определить возможность использования библиотеки LangChain или аналога. Возможно применение локальной LLM типа Llama 2 (через API).

На первом этапе предполагается генерация текстового контента по запросу.

На втором этапе планируется развитие текстовой генерации с учётом контекста, промптинжиниринга и применения многоагентного подхода с помощью библиотеки LangChain или аналога.

На третьем этапе предполагается генерация изображений (в т.ч. логотипов, иконок, иллюстраций).

На четвёртом этапе генерация описания по картинке с учётом текстового контекста проекта.
